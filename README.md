# Selenium-twitter-web-scraper
My first project in python! Webscraper that uses selenium to extract data from twitter based off a given hashtag and time period.

Dependencies:
-Selenium
-Undetected chrome driver

How to use:
-Install dependencies
-Change the variable "hashtag" to the target hashtag
-Change the variable "start_date" to the date that represents the first day within a given time period
-Change the variable "end_date" to the date that represents the last day wihin a given time period
-Open terminal and run: python scraper.py
-Login to twitter account (ideally burner)
-Wait for process
-Save .csv file
